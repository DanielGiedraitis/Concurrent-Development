To set up this project on your system, follow these steps\+:


\begin{DoxyEnumerate}
\item Ensure that GCC is installed on your machine.
\item Clone this repository to your local environment.
\item Open your terminal and navigate to the project directory.
\item Once in the folder, use the following commands\+:
\begin{DoxyItemize}
\item {\bfseries{Build the Executables\+:}}
\begin{DoxyItemize}
\item Run {\ttfamily make} or {\ttfamily make build} to generate the main executables.
\item For {\ttfamily rendezvous}\+:
\begin{DoxyItemize}
\item Execute {\ttfamily make rendezvous}.
\end{DoxyItemize}
\item To create a debug version with symbols\+:
\begin{DoxyItemize}
\item Use {\ttfamily make debug} to build the executables with debug symbols.
\end{DoxyItemize}
\end{DoxyItemize}
\item {\bfseries{Run the Executables\+:}}
\begin{DoxyItemize}
\item To execute the generated programs, use the shell command {\ttfamily ./rendezvous}
