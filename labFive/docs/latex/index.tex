This project revolves around solving the Dining Philosophers problem, where multiple philosophers compete for a limited number of resources (in this case, dining utensils) in a way that prevents deadlock and starvation.\hypertarget{index_autotoc_md3}{}\doxysection{\texorpdfstring{List of Files and Contents}{List of Files and Contents}}\label{index_autotoc_md3}

\item {\bfseries{\doxylink{main_8cpp}{main.\+cpp} -\/}} The file containing the main function. This main function creates a solution to the Dining Philosophers problem.

To set up this project on your system, follow these steps\+:


\begin{DoxyEnumerate}
\item Ensure that GCC is installed on your machine.
\item Clone this repository to your local environment.
\item Open your terminal and navigate to the project directory.
\item Once in the folder, use the following commands\+:
\begin{DoxyItemize}
\item {\bfseries{Build the Executables\+:}}
\begin{DoxyItemize}
\item Run {\ttfamily make} or {\ttfamily make build} to generate the main executables.
\item For {\ttfamily philosophers}\+:
\begin{DoxyItemize}
\item Execute {\ttfamily make philosophers}.
\end{DoxyItemize}
\item To create a debug version with symbols\+:
\begin{DoxyItemize}
\item Use {\ttfamily make debug} to build the executables with debug symbols.
\end{DoxyItemize}
\end{DoxyItemize}
\item {\bfseries{Run the Executables\+:}}
\begin{DoxyItemize}
\item To execute the generated programs, use the shell command {\ttfamily ./philosophers}
\end{DoxyItemize}
\end{DoxyItemize}
