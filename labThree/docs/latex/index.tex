\begin{DoxyItemize}

\item {\bfseries{\doxylink{_semaphore_8cpp}{Semaphore.\+cpp} -\/}} Implementation file for the \doxylink{class_semaphore}{Semaphore} class.
\item {\bfseries{\doxylink{_barrier_8cpp}{Barrier.\+cpp} -\/}} The implementation file for the \doxylink{class_barrier}{Barrier} class.

\item {\bfseries{\doxylink{main_8cpp}{main.\+cpp} -\/}} The file containing the main function. This main function will create at least two threads demonstrating a reusable barrier in action.

\item {\bfseries{README -\/}} Contains Lab 3 information, installation instructions, compilation, and execution guidelines.

To set up this project on your system, follow these steps\+:


\begin{DoxyEnumerate}
\item Ensure that GCC is installed on your machine.
\item Clone this repository to your local environment.
\item Open your terminal and navigate to the project directory.
\item Once in the folder, use the following commands\+:
\begin{DoxyItemize}
\item {\bfseries{Build the Executables\+:}}
\begin{DoxyItemize}
\item Run {\ttfamily make} or {\ttfamily make build} to generate the main executables.
\item For {\ttfamily barrier}\+:
\begin{DoxyItemize}
\item Execute {\ttfamily make barrier}.
\end{DoxyItemize}
\item To create a debug version with symbols\+:
\begin{DoxyItemize}
\item Use {\ttfamily make debug} to build the executables with debug symbols.
\end{DoxyItemize}
\end{DoxyItemize}
\item {\bfseries{Run the Executables\+:}}
\begin{DoxyItemize}
\item To execute the generated programs, use the shell command {\ttfamily ./barrier}
\end{DoxyItemize}
\end{DoxyItemize}
